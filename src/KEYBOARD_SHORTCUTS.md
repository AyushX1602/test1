# ⌨️ Keyboard Shortcuts Guide

## Global Shortcuts

### Command Palette
**⌘ K** (Mac) or **Ctrl K** (Windows/Linux)  
Opens the global command palette for quick navigation and actions.

**ESC**  
Closes the command palette or any open modal.

---

## Command Palette Navigation

When the command palette is open:

**↑ (Up Arrow)**  
Navigate to the previous item in the list.

**↓ (Down Arrow)**  
Navigate to the next item in the list.

**↵ (Enter)**  
Execute the selected command or navigate to the selected page.

**ESC**  
Close the command palette.

---

## Available Commands

### Pages
- **Dashboard Overview** - Navigate to the main dashboard
- **Analytics** - View detailed analytics and charts
- **Products** - See product performance reports
- **Settings** - Manage account and preferences
- **Support** - Access help and documentation

### Actions
- **Export Report** - Download current data as PDF/CSV/Excel
- **Open Filters** - Show filtering options

### Shortcuts
- **Toggle Theme** - Switch between light and dark mode

---

## Search Tips

The command palette supports **fuzzy search**:
- Type partial words: "anal" matches "Analytics"
- Use keywords: "dark" matches "Switch to Dark Mode"
- Try categories: "page" shows all page navigation commands

---

## Pro Tips

1. **Quick Navigation**: Use ⌘K → type first letter → Enter
   - ⌘K → "a" → Enter = Navigate to Analytics
   
2. **Theme Toggle**: ⌘K → "dark" → Enter = Toggle theme

3. **Export Data**: ⌘K → "export" → Enter = Open export menu

4. **Mouse Alternative**: Click the ⌘ icon in the top-right corner

---

## Visual Indicators

- **Purple highlight** = Selected item
- **Purple icon** = Currently selected action
- **Gray icons** = Unselected items
- **↵ badge** = Confirmation hint
